# Contributing to Vermont Political Action Tracker

Thanks for checking! I welcome corrections, additions, and cross-references from other researchers.

## How to Contribute

### Small Fixes
Suggest edits by clicking the ✏️ pencil icon on any markdown page (no Git experience needed).

### Substantial Changes
Please open an issue first to discuss scope before submitting a PR.

### Fact Verification
Cross-referencing with LegiScan VT or official legislature.vt.gov records appreciated. Please link sources!

## Attribution Format

When contributing material from outside research:
> Source: [Publication/Public Body], Date Accessed, URL

---

All contributions acknowledged in commit histories and annual summaries.
