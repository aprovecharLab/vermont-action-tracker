# 📋 Monthly Workflow Checklist — Vermont Action Tracker

## 1. Research & Draft (~20 min)

- [ ] Pull latest bill data from [LegiScan VT](https://legiscan.com/VT)
- [ ] Check legislature.vermont.gov for committee/hearing updates
- [ ] Scan VTDigger / Seven Days for relevant coverage
- [ ] Copy blank template into `/YEAR/` folder
- [ ] Rename: `YYYY-MM-monthname.md` (e.g., `2026-08-august.md`)

## 2. Fill In Content (~20 min)

- [ ] Replace `[MONTH YEAR]` in title
- [ ] Update each DSA category section with new bill findings
- [ ] Delete empty category subsections (keeps doc lean)
- [ ] Fill in summary table ("Where We Stand")
- [ ] Write path forward recommendations
- [ ] Update action footer table with this month's priorities
- [ ] Replace `[DATE]` in footer line

## 3. Review & Commit (~5 min)

- [ ] Check all `../CIVIC_ACTION_TOOLKIT.md` links resolve
- [ ] Verify no leftover placeholder text (brackets, blanks)
- [ ] Commit with message: `"YYYY-MM monthname tracker"`

## 4. Update README (~5 min)

- [ ] Move previous month to archive table row
- [ ] Add new month as "Current Month" with summary snapshot
- [ ] Update key focus areas section
- [ ] Commit with message: `"Update README — YYYY-MM monthname"`

## 5. Post to Bluesky (~5 min)

- [ ] Screenshot 1–4 key sections from rendered markdown page
- [ ] Compose post: brief intro + key items + GitHub link
- [ ] Attach screenshots
- [ ] Include link to both monthly file and CIVIC_ACTION_TOOLKIT.md
- [ ] Post and pin to profile

---

**Total time: ~55 minutes/month**

*Tip: Save this checklist as a file in your repo root called `MONTHLY_CHECKLIST.md` so it's always one click away.*
